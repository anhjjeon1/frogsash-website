사용설명서
메트로 R&S AI v21.0 사용설명서
최초 설정 (1회)
· 좌측 상단 설정 버튼 탭
· Script URL / Sheet ID / 시트명은 자동 입력됨
· 작업자 이름만 A / B / C 입력 후 저장

데이터 준비 (A 작업자, 최초 1회)
· '엑셀 파일 → 구글시트 업로드' 탭 → 원본 엑셀 선택
· 엑셀 데이터가 구글시트에 자동 업로드됨

데이터 불러오기 (A/B/C 작업자)
· '구글시트에서 불러오기' 탭 → 하자리스트 로드
· 동/호 숫자 입력 → 해당 세대 하자 자동 필터
· 우측 마이크 버튼: 음성으로 숫자 입력 가능
· '210' 처럼 앞 0 없이 입력해도 '0210'과 일치

사진 촬영 (자동 저장)
· 수리전/수리후/확인서 탭 → 사진 촬영
· 촬영 즉시 Google Drive 업로드 + 구글시트 셀에 이미지 자동 표시
· 다른 작업자는 '새로고침'으로 최신 사진 확인

하단 버튼
· 엑셀저장: 사진 포함 엑셀 파일 폰에 다운로드
· PDF저장: 브라우저 인쇄 → 'PDF로 저장' 선택 (내 파일 → 다운로드)
· 새로고침: 구글시트에서 최신 데이터 동기화
· 전체삭제: 로컬 모든 데이터 삭제 (확인 팝업 있음)

마이그레이션 (v17 이하 이력 있는 경우)
· 설정 패널 하단 '기존 사진 → Drive 이미지 마이그레이션' 버튼
· 구버전에 base64로 저장된 사진을 Drive URL + =IMAGE() 수식으로 일괄 변환
· 건 로드됨
